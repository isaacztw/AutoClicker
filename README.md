# AutoClicker

A very simple autoclicker project, designed to automate actions (for those who aren't keen on clicking, haha).

-----------------------------------------------------------------------------------------------------------------------------------------

For this project to work, you need to install PyAutoGUI and Keyboard (in the terminal). PyAutoGUI is a Python library used to automate computer actions, simulating human interaction with a mouse and keyboard.

Install (in terminal):

pip install pyautogui
pip install keyboard

-----------------------------------------------------------------------------------------------------------------------------------------

Initially, I thought of creating this project to use in games that depend on clicking to progress, and this greatly facilitated my progress, haha.
